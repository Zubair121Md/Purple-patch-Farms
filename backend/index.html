Excel Upload & Data Import

Excel Upload Instructions

Upload an Excel file with the following columns:

  • Month: Format like "24-Apr" or "2024-04"
  • Particulars: Product name (e.g., "Arugula (Rocket Lettuce)")
  • Type: "In-house" or "Outsourced"
  • Inward Quantity: Quantity received/purchased (in kg)
  • Inward Eff. Rate: Rate per kg for inward quantity
  • Inward Value: Total value of inward quantity
  • Outward Quantity: Quantity sold (in kg)
  • Outward Eff. Rate: Selling rate per kg
  • Outward Value: Total sales value

Auto-Detection: The system will automatically detect inhouse production (when outward > inward) and wastage (when inward > outward).

P&L Upload & Cost Allocation

P&L Upload Instructions

Upload a Purple Patch P&L Excel file with the following format:

  • Column A: Particulars (e.g., "Cultivation Expenses", "Office Expenses")
  • Column B: Amounts (e.g., "211658.45", "50000")
  • Period: Should contain date range like "1-Apr-24 to 30-Apr-24"

Allocation Logic:

  • I Items: 100% allocated to Inhouse products
  • O Items: 100% allocated to Outsourced products
  • B Items: Split by calculated ratio (default: 18.22% Inhouse, 81.78% Outsourced)
